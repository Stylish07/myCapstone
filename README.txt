안녕하세요 교수님. 2조 조장 윤현식 입니다.

저희가 아직 어플리케이션을 완성하진 못해서 현재까지 구현된 부분을 정리해서 보냅니다.

저희는 우선 안드로이드 어플리케이션을 만들었고 응급상황에 대비해 주변 응급실 찾기(아직 서울시 한정), 응급실 현황 보기 등의 기능을 우선 구현했습니다.
다만 아직 디자인 적인 부분이나 완성도 부분에서 부족합니다. 이부분은 차차 공부를 더 해가면서 추가해보도록 하겠습니다.

app - release 폴더에 capstone1217.apk 파일이 있습니다. 빌드 완료된 apk 파일입니다. 앱 다운하실때 이것을 이용하셔도 됩니다. 
감사합니다.

사용된 것들:

서울시 열린 데이터광장 - 서울특별시 응급실 위치 정보 API (https://data.seoul.go.kr/dataList/OA-20338/S/1/datasetView.do)

응급실 실시간 조회 http://portal.nemc.or.kr/medi_info/dashboards/dash_total_emer_org_popup_for_egen.do
